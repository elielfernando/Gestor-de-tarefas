# Gestor de Tarefas
link do projeto https://gestaodetarefas-505c9.web.app/tasklist

Um aplicativo simples de gerenciamento de tarefas desenvolvido com Angular.

## Funcionalidades
- Listar tarefas
- Visualizar detalhes de tarefas
- Filtrar tarefas
- Adicionar, editar e remover tarefas

## Tecnologias
- Angular-framework front-end moderno
- Firebase (para hospedagem)
- TypeScript – tipagem estática e segurança no código

## Como executar
1. Instale as dependências: `npm install`
2. Inicie o servidor de desenvolvimento: `ng serve`
3. Abra o navegador em `http://localhost:4200`

## Build
Para build de produção: `ng build`

## Testes
Execute testes unitários: `ng test`

```bash
ng e2e
```

Angular CLI does not come with an end-to-end testing framework by default. You can choose one that suits your needs.
