# 📌 Sistema de Gestão de Tarefas

## 🧠 Sobre o projeto
Aplicação web para gerenciamento de tarefas, permitindo criar, editar e concluir atividades.

## 🚀 Tecnologias
- Angular
- TypeScript
- LocalStorage

## 💻 Funcionalidades
- Criar tarefas
- Editar tarefas
- Marcar como concluída
- Persistência no navegador
